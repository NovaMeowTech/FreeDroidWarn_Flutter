# Changelog

## 1.0.0

* Initial release.
* Implements `FreeDroidWarn.showWarningOnUpgrade(int buildVersion)` on Android.
* Displays an alert dialog warning users about Google's upcoming developer
  verification requirement (starting 2026/2027).
* Includes translations for: Arabic, Czech, German, Greek, Spanish, French,
  Hindi, Hungarian, Indonesian, Italian, Japanese, Korean, Dutch, Polish,
  Portuguese, Russian, Slovak, Swedish, and Turkish.
